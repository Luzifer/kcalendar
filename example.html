To use this calendar simply create a div which got an unique id. Then use the following JavaScript-Code to display the calendar: 

<script type="text/javascript">
	ShowCurrentCalendar(id of the div, name of the callback function);
</script>
The callback function have to accept three parameters: year, month, day.

Here is the example used in this page to display the calendar and a message box when the user clicks a date: 

<script type="text/javascript">
	function OnCalendarClick(year, month, day) {
		alert('You clicked on this date: ' + year + '-' + month + '-' + day);
	}
	ShowCurrentCalendar('kalender', 'OnCalendarClick');
</script>
